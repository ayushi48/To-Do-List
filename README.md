# 📝 To-Do List Application

A modern, interactive, and feature-rich To-Do List web application built using React.js and Tailwind CSS, designed to help users organize tasks, manage time efficiently, and boost productivity with calendars, reminders, and focus tools.

## 📌 Project Overview

The To-Do List Application is a personal productivity tool enabling users to efficiently add, track, and manage tasks. It offers multiple views including:

- All Tasks
- Calendar View
- Today’s Reminders

Additionally, the app includes focus-enhancing features such as a stopwatch/timer and UI customization options.

All data is stored in the browser’s Local Storage, ensuring tasks persist even after page refreshes or browser restarts, without the need for a backend or database.

## 🛠️ Tech Stack

- React.js – Build dynamic and interactive UIs
- Tailwind CSS – Create modern, responsive, and clean UI styling
- Local Storage – Persistent client-side task storage

## ⭐ Key Highlights

- 100% Frontend-Only: No backend, database, or authentication required
- Persistent Data: Tasks, status, importance, and settings saved in Local Storage
- User-Focused Design: Clean UI with customizable settings

## ✨ Features

### ✅ Task Management
- Add, update, and delete tasks easily
- Mark tasks as completed or important
- Filter tasks based on:
  - All tasks
  - Completed tasks
  - Important tasks
  - Pending tasks

### 📅 Calendar Integration
- Monthly calendar view
- Agenda-style list view
- Tasks auto-populate on respective dates
- Tasks sorted by date and importance
- Easy navigation between months, weeks, and days

### ⏰ Reminders & Focus Tools
- Today’s Reminders panel
- Built-in stopwatch / timer for productivity
- Quick overview of:
  - Total tasks
  - Completed tasks
  - Important tasks
  - Remaining tasks

### 🎨 Settings & Customization
- Change background colors and themes
- Smooth navigation between sections
- Clean, colorful, and responsive UI

## 🗂️ Application Sections

| Section      | Description |
| ------------ | ----------- |
| Home         | Overview and daily motivation |
| Add Task     | Create and schedule tasks |
| Focus        | Timer / stopwatch for productivity |
| All Tasks    | Manage, filter, and update tasks |
| Calendar     | Date-wise task visualization |
| Settings     | UI customization options |

## 💾 Local Storage (Data Persistence)
- All tasks are saved in the browser Local Storage
- Data remains after:
  - Page refresh
  - Browser restart
  - Offline usage
- Ensures a fast and lightweight solution for task management
